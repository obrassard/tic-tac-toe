# Tic Tac Toe
Jeu de Tic Tac Toe pour l'atelier présenté au CJE Marguerite-d'Youville

### Prérequis

- Node.js (v16 ou plus récent) et NPM (https://nodejs.org/en/)

### Installation

```
npm install
npm start
```
Ouvrir le navigateur à l'adresse affichée dans la console.
